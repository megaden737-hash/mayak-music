# Mayak Music — Обновлённый сайт

**Новая цветовая палитра:** Глубокий тёмно-синий (#0B0F1C), белый, ярко-жёлтый (#FACC15) + акцентный синий (#3B82F6).

**Стиль:** Более минималистичный, но с драйвом и креативом. Чистые карточки, щедрые отступы, сильные заголовки, яркие акценты на кнопках.

## Что добавлено / исправлено

- Много новых новостей (10 шт.)
- Расширен битстор до 18 битов
- 8 готовых треков под ключ
- Улучшенные превью битов
- Консистентная минималистичная навигация и модальные формы на всех страницах
- Много призывов к действию
- Цвета: синий + жёлтый + белый

## Запуск

```bash
cd mayak-music
python3 -m http.server 8000
```

Откройте http://localhost:8000

Все изображения уже сгенерированы и лежат в assets/images.
